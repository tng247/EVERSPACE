# Official TNG Repository: EVERSPACE
## Description
This repository contains additional content for inclusion within the official [EVERSPACE](https://everspace-game.com) product.

Stable redistributable content can be found on the [releases](https://github.com/tng247/EVERSPACE/releases) page, latest in-flight development content can be found on the [master](https://github.com/tng247/EVERSPACE/tree/master) branch.

## Usage
With [Git](https://git-scm.com) installed open a console/terminal window and execute the following command within your EVERSPACE configuration directory:

`git clone https://github.com/tng247/EVERSPACE.git .`

For example:

* **Windows**: `%USERPROFILE%\AppData\Local\RSG\Saved\Config\WindowsNoEditor`
* **Linux**: `~/.config/Epic/RSG/Saved/Config/LinuxNoEditor`

Once complete you will need to perform the following additional steps:

1. Launch EVERSPACE and navigate to **Options** > **Controls** > **Controller**
1. Change **Controller Sensitivity** to **0.0**
1. Disable **Invert Y-Axis**
1. Disable **Centered Crosshair**

> ***Note**: Currently the additional steps are required as some configuration settings are not stored within `.ini` files.*
